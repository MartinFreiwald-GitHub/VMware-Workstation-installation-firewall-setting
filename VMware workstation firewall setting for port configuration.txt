In Mircosoft Windows 10, use the search bar next to the start button in the footer bar.  Type Windows Defender Firewald and Advanced Security application.  Select it.  In the left control pane select Inbound Rules Icon.  Scroll down to VMware authd Service.  Click on it to open control panel.   Select the Ports and Protocol tab.  Click on the Specific Ports draw down window.  In the field box type 80,443 and apply.  Now, view the local port column in the Windows Defender Firewald and Advanced Security application to see both port 80 and 443 TCP are selected for filter only in the, on operation. Close the Windows Defender Firewald and Advanced Security application.  Start using your VMware Workstation.
